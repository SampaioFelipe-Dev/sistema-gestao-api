# Kore Manager

> Sistema completo de e-commerce e gestão comercial.

## 💻 Sobre o Projeto

O **Kore Manager** é uma aplicação Full-Stack desenvolvida para facilitar o controle de vendas, estoque e a gestão diária de negócios. O sistema foi projetado com foco em performance e escalabilidade, oferecendo uma interface intuitiva para o usuário final e um painel administrativo robusto para o controle da operação.

Este projeto está em desenvolvimento contínuo e serve tanto como uma solução comercial quanto como um demonstrativo prático de arquitetura de software e desenvolvimento web.

## 🚀 Tecnologias Utilizadas

O ecossistema do projeto foi construído com as seguintes ferramentas:

**Back-end & Infraestrutura:**
* [Node.js](https://nodejs.org/)
* [Express](https://expressjs.com/)
* [Drizzle ORM](https://orm.drizzle.team/)
* [AWS](https://aws.amazon.com/) (Hospedagem e Banco de Dados)

**Front-end:**
* HTML5 / CSS3
* JavaScript (Vanilla)
